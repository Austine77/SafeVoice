3. After Render creates your services, confirm the actual service URLs. If Render gives a different URL, update:
   - worldwide/.env production value
   - backend CLIENT_URL, WORLDWIDE_URL, CORS_ORIGINS
   - meta API base values in worldwide/index.html, staff.html, police-contact.html, social-worker-contact.html if needed.
